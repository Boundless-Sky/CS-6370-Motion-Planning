::How to run::
in terminal run: python simulation.py

::Variables to change (simulation.py)::
multi_sim: Bool -> single or multiple simulation
crowd_size: integer multiple of spawn points
ntests: integer -> how many tests to run
test_cases: path and file name of test cases
fps: Limit frame per seconds to have smooth video
vis_update: how fast to update graphics
obedience: percentage from 0 to 100%
